# Shop Template Project

### Описание

Небольшой шаблон интернет-магазина

### Использовал технологии:
- Frontend: HTML/CSS/Bootstrap
- Backend: Django
- REST API: Django Rest Framework
- Databases: SQL: PostgreSQL | NoSQL: Redis
- Queue: Celery | Broker: RabbitMQ
